# Russia VPN

Закрытый VPN-клиент для Windows и Android. Репозиторий предназначен для публикации готовых файлов установки, а не для open source-разработки.

## Что входит в релиз

- `Russia VPN Setup 1.0.1.exe` - установщик для Windows.
- `app-debug.apk` - Android-приложение.
- `README.md` - инструкция по установке и использованию.

Исходный код, приватные ключи, серверные настройки и внутренние инструменты не публикуются.

## Возможности

- Импорт защищенных ключей формата `russvpn://v1...`.
- Поддержка Windows и Android.
- Режим TUN для проксирования трафика через системный VPN.
- DNS-настройки.
- Split tunneling: исключение выбранных приложений из VPN.
- Маскировка VPN-трафика через TLS/uTLS и совместимые transport-настройки профиля.
- Проверка некорректных Reality-конфигов: ссылки с пустым `pbk=` не принимаются.

## Установка на Windows

1. Скачайте `Russia VPN Setup 1.0.1.exe`.
2. Запустите установщик.
3. Откройте Russia VPN.
4. Перейдите в раздел профилей.
5. Вставьте защищенный ключ `russvpn://v1...`.
6. Нажмите кнопку подключения.



## Установка на Android

1. Скачайте `app-debug.apk` на телефон.
2. Разрешите установку APK из неизвестных источников, если Android попросит.
3. Установите приложение.
4. Откройте Russia VPN.
5. Импортируйте защищенный ключ `russvpn://v1...`.
6. При первом подключении разрешите создание VPN-подключения.

Android-версия использует системный `VpnService`, поэтому после подключения трафик телефона идет через VPN/TUN.
